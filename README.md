# SEPA量化实盘系统 (A股)

基于马克·米勒维尼(Mark Minervini)SEPA策略的A股量化交易系统

## 系统架构

```
数据层 → 筛选层 → 催化层 → 入场层 → 风控层 → 执行层
```

## 核心模块

| 模块 | 文件 | 功能 |
|------|------|------|
| 环境检查 | 00_环境检查.py | 依赖包/接口/目录 |
| 数据引擎 | 01_data_fetcher.py | 腾讯/新浪实时行情+历史K线 |
| 10层筛选器 | 02_stock_filter.py | SEPA趋势+基本面+VCP+催化剂 |
| 风控模块 | 03_risk_manager.py | 凯利公式+三层止损止盈 |
| VCP形态 | 04_vcp_detector.py | 成交量收缩形态识别 |
| 回测引擎 | 05_backtest_engine.py | 2020-2026年历史回测 |

## 快速开始

```bash
cd ~/SEPA_Quant_System_Pro

# 1. 环境检查
python3 main.py check

# 2. 下载数据（全量，首次运行）
python3 01_data_fetcher.py

# 3. 运行选股筛选
python3 02_stock_filter.py

# 4. 查看VCP形态
python3 04_vcp_detector.py 300750

# 5. 运行回测
python3 05_backtest_engine.py
```

## 10层筛选逻辑

- 层级1: 基础过滤(ST/次新股/涨跌停)
- 层级2: SEPA趋势模板(4层金牌标准)
- 层级3: 基本面过滤(营收/净利/ROE)
- 层级4: 成交量确认(放量突破)
- 层级5: 催化剂检测(净利润断层/分析师上调)

## 风控规则

- 凯利公式: 半仓~四分之一凯利
- 止损: 初始-8%，移动止损-8%
- 止盈: +20%→+10%止损, +30%→减仓30%, +50%→全清

## ⚠️ 风险提示

- 本系统仅供研究参考，不构成投资建议
- 历史回测表现不代表未来收益
- A股市场受政策影响大，请谨慎评估
- 实盘前请先进行模拟盘验证
