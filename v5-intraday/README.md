# SEPA 量化交易系统 - v5.0 日内版

> 今天买，明天卖。跑一个月看盈亏率。

## v5.0 核心策略

**T+1 日内交易**：每天14:40尾盘买入，次日14:30全部卖出。

### 六因子评分模型

| 因子 | 权重 | 说明 |
|------|------|------|
| 资金流向 | 25% | 涨幅2-4%最优区间 |
| 历史动量 | 25% | 20日涨幅排名（有趋势未过热） |
| 技术面 | 20% | MA5>MA10 + RSI + MA20支撑 |
| 成交额 | 15% | 资金体量确认 |
| 量比 | 10% | 放量突破确认 |
| 板块动量 | 5% | 涨幅×成交额 |

### 核心改进（相比v4.2）

1. **全A股股票池** — 东方财富API获取5000+只，不再限创业板+科创板
2. **历史动量因子** — 20日涨幅排名，追强不追高
3. **板块分散** — 单行业≤25%仓位，防止ALL IN同一板块
4. **大盘情绪仓位** — 沪深300涨跌调节仓位（强势满仓/弱势30%）
5. **T+1强制卖出** — 次日14:30全部清仓，不恋战
6. **日内止损3%** — 防黑天鹅

### 大盘情绪仓位调节

| 沪深300涨幅 | 情绪 | 仓位比例 |
|------------|------|---------|
| > +0.5% | 强势🔥 | 100% |
| -0.5% ~ +0.5% | 中性⚖️ | 70% |
| -1.5% ~ -0.5% | 弱势⚠️ | 30% |
| < -1.5% | 极弱🛑 | 不买入 |

### 卖出规则

- **T+1到期**：次日14:30强制卖出（核心规则）
- **盈利止盈**：盈利≥8%主动止盈
- **日内止损**：亏损≥3%止损（防黑天鹅）

## 部署

### 服务器

```bash
# 1. 上传脚本
scp daily_sim_trade_v5.py root@82.157.118.198:/opt/stock-monitor/

# 2. 初始化
cd /opt/stock-monitor && venv/bin/python3 daily_sim_trade_v5.py --init

# 3. 设置Crontab
crontab -e
# 添加：
# 30 14 * * 1-5 cd /opt/stock-monitor && /opt/stock-monitor/venv/bin/python3 daily_sim_trade_v5.py --sell >> data/logs/trade_v5_$(date +\%Y\%m\%d).log 2>&1
# 40 14 * * 1-5 cd /opt/stock-monitor && /opt/stock-monitor/venv/bin/python3 daily_sim_trade_v5.py --buy >> data/logs/trade_v5_$(date +\%Y\%m\%d).log 2>&1
```

### 手动测试

```bash
# 查看状态
python3 daily_sim_trade_v5.py --status

# 手动卖出
python3 daily_sim_trade_v5.py --sell

# 手动买入
python3 daily_sim_trade_v5.py --buy

# 重置系统
python3 daily_sim_trade_v5.py --init
```

## 数据文件

| 文件 | 说明 |
|------|------|
| `data/sim_positions_v5.json` | 持仓+历史记录 |
| `data/daily_report_v5.json` | 每日报告 |
| `data/logs/trade_v5_YYYYMMDD.log` | 交易日志 |

## 数据源

- **东方财富**：全A股列表 + 行业分类 + 实时行情（1次API调用）
- **新浪财经**：K线数据（MA/RSI/动量计算，仅候选股）
